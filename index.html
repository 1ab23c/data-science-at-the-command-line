Data Science at the Command Line

In this practical guide, you'll learn how to leverage the power of the command line for doing data science. By combining small, yet powerful, command-line tools, you can quickly obtain, scrub, explore, and model your data. Even if you're already comfortable processing data with R or Python, being able to integrate the command line into your existing workflow will make you a more efficient and productive data scientist. Setting up the Data Science Toolbox for the Webcast

First, make sure that you have both Virtualbox and Vagrant (version 1.6+) installed. Then, in order to download and launch the Data Science Toolbox, you run the following commands in your terminal / command prompt:

$ vagrant init data-science-toolbox/data-science-at-the-command-line
$ vagrant up
